\# AI Job Market Analysis using MySQL



\## Project Overview

This project analyzes global AI job salary trends, experience-level demand, remote work distribution, and company hiring patterns using SQL queries.



The objective is to extract meaningful business insights from structured AI job datasets.



\## Tools Used

MySQL

Structured Query Language (SQL)



\## Dataset Features

\- Work Year

\- Experience Level

\- Employment Type

\- Job Title

\- Salary (USD)

\- Employee Residence

\- Remote Ratio

\- Company Location

\- Company Size



\## Key Business Questions Answered



1\. Which AI roles have the highest salaries?

2\. How salaries vary across experience levels?

3\. Which company locations offer the highest-paying jobs?

4\. Are remote roles higher paying than onsite roles?

5\. Which company sizes hire the most AI professionals?

6\. How salaries changed over recent years?

7\. Entry-level job availability in AI roles



\## Example Insights



\- Senior and expert roles command the highest salaries

\- Remote opportunities remain competitive with onsite roles

\- Large companies dominate hiring volume

\- AI salaries show steady growth trends over recent years
